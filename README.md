# Anime Squadron Guide Hub

Next.js static site for `anime-squadron.net`.

## Stack

- Next.js App Router
- TypeScript
- Static export (`output: 'export'`) for Cloudflare Pages-compatible hosting

## Routes

- `/`
- `/codes/`
- `/tier-list/`
- `/traits/`
- `/beginner-guide/`
- `/privacy/`, `/terms/`, `/contact/` are noindex utility pages

## Trust boundary

This is an unofficial fan-made guide. Do not add exact codes, unit rankings, trait values, or reward details without source/in-game verification.

## Commands

```bash
npm run lint
npm run build
npm run check:content
```
